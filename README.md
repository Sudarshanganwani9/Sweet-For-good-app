# 🍬 Sweet For Good

A modern web application designed to promote and manage sweet distribution for social causes. This platform allows users to contribute, organize, and track sweet donation activities efficiently.

## 🚀 Features

- User-friendly interface
- Create and manage donation events
- Track contributions and participation
- Responsive design for mobile and desktop
- Secure and scalable backend integration

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js / Express.js
- Database: MongoDB / (your DB)
- Styling: CSS / Tailwind (update based on your project)
